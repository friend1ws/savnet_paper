# Cancer Gene Census
1. Download cancer gene census data from 
2. Upload to the directory CancerGeneCensus, modify the input file path in the make_cancer_gene_census.py,
and do `python make_cancer_gene_census.py`

# Lawrence et al.
1. Download the Supplementary Table 2 and 3 of Nature 2014 paper (doi:10.1038/nature12912).
2. Upload them to the directory `LawrenceEtAl`, and do `python make_LawrenceEtAl_sig_gene.py`
